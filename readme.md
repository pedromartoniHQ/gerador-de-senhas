# 🔐 Gerador de Senhas Seguras em Python

Projeto em Python para geração de senhas aleatórias e seguras, utilizando letras maiúsculas, minúsculas, números e caracteres especiais.

## 📌 Objetivo do projeto
Este projeto tem como objetivo praticar:
- Lógica de programação em Python
- Boas práticas de segurança relacionadas a senhas
- Organização e estruturação de código
- Tratamento de erros de entrada do usuário



## ⚙️ Funcionalidades
- Geração de senhas aleatórias
- Definição do tamanho da senha pelo usuário
- Uso de caracteres seguros (letras, números e símbolos)
- Validação de entrada para evitar erros
- Código organizado e reutilizável

---

## 🚀 Tecnologias Utilizadas
- Python 3
- Bibliotecas padrão (`random`, `string`)
